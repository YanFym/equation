# README

Простое приложение для подсчёта квадратных уровнений и не только.

# Требования 

* Ruby (version 2.6.3)

* Node.js

# УСТАНОВКА

* Клонировать репозиторий

* bundle install

* yarn install

* rails db:create, rails db:migrate

# ЗАПУСК

* rails s

* перейти на http://localhost:3000
